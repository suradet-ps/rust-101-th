# 📖 Glossary: Rust-101 (Thai Edition)

เอกสารนี้รวบรวมคำศัพท์เทคนิคภาษาไทยที่ใช้ในโครงการแปล Rust-101 เพื่อความสม่ำเสมอ (Consistency) และความถูกต้องแม่นยำ (Accuracy)

## 🟢 Core Concepts (หัวใจหลักของ Rust)

| English Term | Thai Translation | Description / Notes |
| :--- | :--- | :--- |
| **Ownership** | ความเป็นเจ้าของ | แนวคิดหลักที่ใช้จัดการหน่วยความจำของ Rust |
| **Borrowing** | การยืม | การเข้าถึงข้อมูลโดยไม่ยึดความเป็นเจ้าของ (ผ่าน Reference) |
| **Lifetime** | อายุขัย | ช่วงเวลาที่ Reference สามารถใช้งานได้อย่างปลอดภัย |
| **Move** | การย้าย | การโอนความเป็นเจ้าของข้อมูลจากตัวแปรหนึ่งไปอีกตัวแปรหนึ่ง |
| **Copy** | การคัดลอก | การทำสำเนาข้อมูล (สำหรับประเภทข้อมูลที่เก็บใน Stack ได้) |
| **Clone** | การโคลน | การทำสำเนาข้อมูลแบบ Deep Copy (มักใช้กับข้อมูลใน Heap) |
| **Mutability** | ความเปลี่ยนแปลงได้ | สถานะของตัวแปรที่สามารถแก้ไขค่าได้ |
| **Immutability** | ความคงตัว / เปลี่ยนแปลงไม่ได้ | สถานะตั้งต้นของตัวแปรใน Rust ที่แก้ไขค่าไม่ได้ |

## 📦 Data Structures & Types (โครงสร้างข้อมูลและชนิดตัวแปร)

| English Term | Thai Translation | Description / Notes |
| :--- | :--- | :--- |
| **Struct** | สตรัค (โครงสร้างข้อมูล) | ใช้ทับศัพท์ "สตรัค" ในบริบทโค้ด หรือ "โครงสร้างข้อมูล" ในบริบททั่วไป |
| **Enum** | อีนัม | ชนิดข้อมูลที่เก็บค่าได้หนึ่งค่าจากชุดตัวเลือกที่กำหนด |
| **Trait** | เทรต | ชุดพฤติกรรมที่กำหนดให้กับ Type (คล้าย Interface ในภาษาอื่น) |
| **Vector** | เวกเตอร์ | อาเรย์ที่ยืดหยุ่นขนาดได้ |
| **Slice** | สไลซ์ | การอ้างอิงไปยังส่วนหนึ่งของคอลเลกชัน (เช่น ข้อมูลบางส่วนใน Array) |
| **String** | สตริง (ข้อความ) | ใช้ทับศัพท์เมื่อพูดถึง Type `String`, ใช้ "ข้อความ" ทั่วไป |
| **Tuple** | ทูเพิล | กลุ่มของค่าที่มีชนิดข้อมูลต่างกันได้ และมีจำนวนสมาชิกคงที่ |
| **Smart Pointer** | สมาร์ทพอยเตอร์ | ตัวชี้ที่มีความสามารถเพิ่มเติมในการจัดการข้อมูล (เช่น `Box`, `Rc`) |
| **Generic** | เจเนอริก | ชนิดข้อมูลทั่วไปที่สามารถแทนที่ด้วยชนิดข้อมูลใดก็ได้ |

## ⚙️ Control Flow & Functions (การควบคุมและการทำงาน)

| English Term | Thai Translation | Description / Notes |
| :--- | :--- | :--- |
| **Pattern Matching** | การจับคู่รูปแบบ | กลไกการตรวจสอบโครงสร้างของข้อมูล (ใช้กับ `match`) |
| **Expression** | นิพจน์ | โค้ดที่ประมวลผลแล้วได้ค่าผลลัพธ์ |
| **Statement** | คำสั่ง | โค้ดที่สั่งการทำงานแต่ไม่ได้คืนค่าผลลัพธ์ |
| **Iterator** | ตัววนซ้ำ | อ็อบเจกต์ที่ช่วยให้วนลูปผ่านคอลเลกชันได้ |
| **Closure** | โคลเชอร์ | ฟังก์ชันนิรนามที่สามารถจับค่าจากสิ่งแวดล้อมรอบข้างได้ |
| **Function** | ฟังก์ชัน | ชุดคำสั่งที่ทำงานเฉพาะอย่าง |
| **Method** | เมธอด | ฟังก์ชันที่ผูกอยู่กับ Struct หรือ Enum |
| **Associated Function** | ฟังก์ชันที่เกี่ยวข้อง | ฟังก์ชันที่อยู่ใน `impl` block แต่ไม่ได้ทำงานกับ instance (คล้าย static method) |

## 🛡️ Safety & Error Handling (ความปลอดภัยและการจัดการข้อผิดพลาด)

| English Term | Thai Translation | Description / Notes |
| :--- | :--- | :--- |
| **Panic** | แพนิค | อาการที่โปรแกรมหยุดทำงานทันทีเมื่อเจอข้อผิดพลาดร้ายแรง |
| **Result** | Result (ผลลัพธ์) | Enum ที่ใช้ส่งค่าคืนเมื่อการทำงานอาจสำเร็จหรือล้มเหลว (`Ok` หรือ `Err`) |
| **Option** | Option (ทางเลือก) | Enum ที่ใช้แทนค่าที่อาจจะมีหรือไม่มีอยู่จริง (`Some` หรือ `None`) |
| **Unwrap** | การแกะ (Unwrap) | การดึงค่าข้างใน `Option` หรือ `Result` ออกมา (ถ้าไม่มีค่าจะเกิด Panic) |
| **Unsafe** | อันเซฟ (ไม่ปลอดภัย) | บล็อกโค้ดที่โปรแกรมเมอร์ต้องรับประกันความปลอดภัยเอง |

## 🏗️ Project Structure (โครงสร้างโปรเจกต์)

| English Term | Thai Translation | Description / Notes |
| :--- | :--- | :--- |
| **Crate** | เครต | หน่วยของการคอมไพล์ใน Rust (อาจเป็น Library หรือ Executable) |
| **Module** | โมดูล | พื้นที่จัดเก็บชื่อและฟังก์ชันเพื่อความเป็นระเบียบ |
| **Package** | แพ็กเกจ | ชุดของ Crate หนี่งตัวหรือมากกว่าที่ทำงานร่วมกัน |
| **Cargo.toml** | ไฟล์ Cargo.toml | ไฟล์กำหนดการตั้งค่าและ Dependency ของโปรเจกต์ |
| **Dependency** | ดีเพนเดนซี (ส่วนต่อขยายพึ่งพา) | ไลบรารีภายนอกที่โปรเจกต์ต้องใช้ |

---
**หมายเหตุ:**

1. คำศัพท์ที่มีวงเล็บภาษาอังกฤษกำกับ เช่น "ความเป็นเจ้าของ (Ownership)" ให้ใช้รูปแบบนี้ในการกล่าวถึงครั้งแรก
2. คำทับศัพท์ภาษาอังกฤษ ให้เขียนเป็นภาษาไทยตามตาราง เช่น "เทรต", "สตรัค", "อีนัม"
3. ชื่อเฉพาะของ Type ใน Code เช่น `Option`, `Result`, `String`, `Vec` ให้คงภาษาอังกฤษไว้เสมอ
