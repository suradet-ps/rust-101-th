## ยินดีต้อนรับสู่ Rust-101 (ภาษาไทย)

นี่คือ Rust-101 คู่มือสอนภาษา Rust ฉบับกระชับ ซึ่งตั้งใจออกแบบมาให้เป็นคอร์สแบบ **เน้นลงมือทำจริง** (Interactive hands-on) เพราะผมเชื่อว่าวิธีเดียวที่จะเรียนรู้ภาษาโปรแกรมได้ดีที่สุดคือการเขียนโค้ดจริงๆ ดังนั้น เตรียมตัวเขียนโค้ดไปพร้อมๆ กันตลอดบทเรียนได้เลยครับ

หากคุณมีข้อสงสัยตรงไหนที่ในบทเรียนนี้ไม่ได้อธิบายไว้ ลองดูที่หัวข้อ “แหล่งข้อมูลเพิ่มเติม” (Additional Resources) ด้านล่างนะครับ 

บทเรียนนี้ตั้งสมมติฐานว่าคุณน่าจะมีพื้นฐานการเขียนโปรแกรมมาบ้างแล้ว ดังนั้นผมจะไม่ลงลึกในคอนเซปต์พื้นฐานที่ภาษาส่วนใหญ่มีเหมือนกัน แต่จะมุ่งเน้นไปที่ **ความพิเศษ** ของภาษา Rust เป็นหลักครับ

## ทำไมต้อง Rust?

การที่คุณเข้ามาอ่านหน้านี้ ผมเดาว่าคุณคงตัดสินใจเปิดใจลองดูภาษา Rust บ้างแล้ว ดังนั้นผมคงไม่ต้องโน้มน้าวอะไรมาก แต่เผื่อใครยังลังเล นี่คือเหตุผลว่าทำไมผมถึงคิดว่า Rust เป็นภาษาที่คุ้มค่าแก่การเรียนรู้ครับ

ในปัจจุบัน Rust เป็นภาษาที่มีเป้าหมายที่เป็นเอกลักษณ์มาก มันมุ่งเน้นให้เราสามารถควบคุมหน่วยความจำและพฤติกรรมการทำงานของโปรแกรมได้ในระดับเดียวกับภาษา C++ (เช่น การเลือกใช้ static หรือ dynamic dispatch) ซึ่งทำให้เราสามารถสร้างระบบที่ซับซ้อนโดยไม่มีต้นทุนแฝงตอนรันโปรแกรม (Zero run-time cost)

ความเจ๋งคือ Rust นำความสามารถข้างต้นมาผสานเข้ากับความสะดวกสบายในสไตล์ภาษาเชิงฟังก์ชัน (Functional languages) ระดับสูง และมาพร้อมกับความปลอดภัยที่รับประกันได้ (หมายความว่าโปรแกรมจะไม่แครชหรือพังแบบไร้การควบคุม)

ภาษาโปรแกรมส่วนใหญ่มักต้องแลกอย่างใดอย่างหนึ่ง ระหว่าง **การควบคุม** กับ **ความปลอดภัย** (เช่น ยอมเสียการควบคุมเพื่อความปลอดภัยโดยการใช้ Garbage Collector หรือในทางกลับกัน) แต่ Rust แตกต่างออกไป Rust สามารถทำงานได้โดยไม่ต้องมีการจัดสรรหน่วยความจำแบบไดนามิก (คือไม่ต้องใช้ Heap) หรือแม้กระทั่งทำงานโดยไม่มีระบบปฏิบัติการ (OS) เลยก็ยังได้

อันที่จริงแล้ว Rust ช่วยกำจัดประเภทของบั๊ก (Bug) ได้มากกว่าภาษาที่มี Garbage Collector เสียอีก นอกจากจะป้องกันปัญหาเรื่อง Dangling pointers และ Double-free แล้ว Rust ยังป้องกันปัญหาอย่าง Iterator invalidation และ Data races ได้อีกด้วย และสุดท้าย Rust ยังคอยเก็บกวาดงานให้คุณ โดยมันจะคืนทรัพยากรต่างๆ (ไม่ว่าจะเป็นหน่วยความจำ, File descriptors หรือทรัพยากรอื่นๆ) ทันทีที่คุณเลิกใช้งานพวกมันครับ
